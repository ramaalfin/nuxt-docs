# Analytics & Insights Generation

Raw data is useless without interpretation. AI excels at analyzing datasets, identifying trends, and generating human-readable insights. This chapter covers how to build UIs that transform raw analytics dashboards into intelligent insight engines.

## 1. Natural Language Data Querying ("Chat with your data")

Instead of forcing users to learn complex filter UIs, let them ask questions about their data.

```
User: "Which marketing campaign had the highest ROI last quarter?"
  ↓
LLM (Text-to-SQL): Generates `SELECT campaign_name, roi FROM campaigns...`
  ↓
Database: Executes query → returns `[{ "campaign": "Summer Sale", "roi": 4.2 }]`
  ↓
LLM (Insight Generation): "The Summer Sale was your top performer with a 4.2x ROI."
```

### Implementation (Vercel AI SDK Tool Calling)

```typescript
// server/api/data-chat.post.ts
import { streamText, tool } from 'ai'
import { z } from 'zod'

export default defineEventHandler(async (event) => {
  const { messages } = await readBody(event)

  const result = await streamText({
    model: openai('gpt-4o'),
    system: 'You are a data analyst. Use the tools provided to answer the user\'s questions about their data.',
    messages,
    tools: {
      queryDatabase: tool({
        description: 'Query the analytics database using natural language. Returns JSON data.',
        parameters: z.object({
          metric: z.enum(['revenue', 'users', 'roi', 'conversion_rate']),
          timeframe: z.enum(['today', 'this_week', 'this_month', 'last_quarter']),
          groupBy: z.string().optional()
        }),
        execute: async ({ metric, timeframe, groupBy }) => {
          // Map to actual DB query
          const data = await analyticsDB.query(metric, timeframe, groupBy)
          return data
        }
      })
    }
  })

  return result.toDataStreamResponse()
})
```

## 2. Automated Insight Summaries

Dashboards show charts; AI explains *why* the charts look that way.

```vue
<!-- components/InsightCard.vue -->
<script setup lang="ts">
const props = defineProps<{
  chartData: AnalyticsData[]
  metricName: string
}>()

// Generate insights when data loads
const { data: insights, isPending } = useQuery({
  queryKey: ['insights', props.metricName, props.chartData],
  queryFn: () => $fetch('/api/generate-insights', {
    method: 'POST',
    body: { data: props.chartData, metric: props.metricName }
  })
})
</script>

<template>
  <div class="insight-card">
    <div class="chart-container">
      <LineChart :data="chartData" />
    </div>
    
    <div class="ai-summary">
      <div v-if="isPending" class="skeleton-text" />
      <template v-else>
        <h4>✨ AI Insights</h4>
        <ul>
          <li v-for="(insight, i) in insights" :key="i">{{ insight }}</li>
        </ul>
      </template>
    </div>
  </div>
</template>
```

```typescript
// server/api/generate-insights.post.ts
const systemPrompt = `
You are a data analyst. I will provide you with raw time-series data for a metric.
Analyze the data and provide exactly 3 bullet points:

`
```

## 3. Generative UI for Data (Chart Generation)

Sometimes the best response isn't text — it's a chart generated dynamically by the LLM.

```typescript
// server/api/chat.post.ts
tools: {
  showChart: tool({
    description: 'Display a chart to the user',
    parameters: z.object({
      title: z.string(),
      type: z.enum(['bar', 'line', 'pie']),
      data: z.array(z.object({ label: z.string(), value: z.number() }))
    }),
    execute: async (chartConfig) => {
      // The tool execution returns the config to the client
      // The AI SDK's `useChat` hook intercepts this and allows you to render a UI component
      return chartConfig 
    }
  })
}
```

```vue
<!-- components/Chat.vue -->
<template>
  <div v-for="message in messages" :key="message.id">
    <div v-if="message.role === 'assistant'">
      <!-- Render text content -->
      <p>{{ message.content }}</p>
      
      <!-- Render UI tools if the model called them -->
      <div v-for="invocation in message.toolInvocations" :key="invocation.toolCallId">
        <div v-if="invocation.toolName === 'showChart' && invocation.state === 'result'">
          <DynamicChart :config="invocation.result" />
        </div>
      </div>
    </div>
  </div>
</template>
```

## 4. Anomaly Detection and Alerts

AI is perfect for identifying "unknown unknowns" in massive datasets. Run scheduled batch jobs to scan analytics for anomalies and generate plain-English alert emails.

```typescript
// Prompt for anomaly analysis
`Here is the traffic data for the last 7 days compared to the previous 7 days.
Data: ${trafficDataJSON}

Task: Identify any statistically significant drops or spikes. 
If an anomaly is found, output a JSON object: { "hasAnomaly": true, "severity": "high", "description": "Traffic from mobile devices dropped 45% on Tuesday." }
If no anomaly, output: { "hasAnomaly": false }`
```
