# Search Enhancement (Semantic)

Traditional keyword search frustrates users when their query doesn't use the exact words in a document. AI-powered semantic search understands *intent* and returns relevant results even when no keywords match. This chapter covers progressively enhancing your existing search with semantic capabilities.

## 1. Progressive Enhancement Strategy

Don't replace your existing search — augment it:

```
Phase 1: Add semantic re-ranking to existing keyword results
Phase 2: Add vector search as a parallel lane (hybrid search)  
Phase 3: Add conversational search ("Ask a question, get an answer")
```

This approach is lower risk and allows A/B testing each improvement.

## 2. Semantic Re-ranking of Existing Results

The fastest win: take your existing keyword search results and re-rank them with semantic understanding. No vector database needed.

```typescript
// server/api/search.post.ts
export default defineEventHandler(async (event) => {
  const { query, page = 1 } = await readBody(event)

  // Step 1: Traditional keyword search (your existing implementation)
  const keywordResults = await db.products
    .where('name', 'ilike', `%${query}%`)
    .orWhere('description', 'ilike', `%${query}%`)
    .limit(50) // Fetch more than needed for re-ranking

  if (keywordResults.length === 0) return { results: [], total: 0 }

  // Step 2: Re-rank semantically
  const reranked = await rerankDocuments(
    query,
    keywordResults.map(p => ({ id: p.id, content: `${p.name}: ${p.description}` })),
    20 // Return top 20 re-ranked results
  )

  // Step 3: Paginate the re-ranked results
  const ITEMS_PER_PAGE = 10
  const pageResults = reranked.slice((page - 1) * ITEMS_PER_PAGE, page * ITEMS_PER_PAGE)
  const fullResults = pageResults.map(r => keywordResults.find(p => p.id === r.id)!)

  return { results: fullResults, total: reranked.length }
})
```

## 3. Conversational Search ("Ask" Mode)

Let users ask questions in natural language and get direct answers with citations.

```vue
<!-- components/SearchInterface.vue -->
<script setup lang="ts">
const searchMode = ref<'keyword' | 'ask'>('keyword')
const query = ref('')

// Standard search results
const { data: searchResults, isPending: isSearching } = useQuery({
  queryKey: ['search', query, searchMode],
  queryFn: () => $fetch('/api/search', { method: 'POST', body: { query: query.value } }),
  enabled: computed(() => query.value.length >= 2 && searchMode.value === 'keyword')
})

// RAG-powered conversational answer
const { messages, handleSubmit, isLoading: isAsking } = useChat({ api: '/api/rag-search' })
const aiAnswer = computed(() => messages.value.find(m => m.role === 'assistant')?.content)
</script>

<template>
  <div class="search-interface">
    <!-- Mode toggle -->
    <div class="search-mode-toggle" role="tablist">
      <button
        role="tab"
        :aria-selected="searchMode === 'keyword'"
        @click="searchMode = 'keyword'"
      >🔍 Search</button>
      <button
        role="tab"
        :aria-selected="searchMode === 'ask'"
        @click="searchMode = 'ask'"
      >✨ Ask AI</button>
    </div>

    <div class="search-input-wrapper">
      <input
        v-model="query"
        :placeholder="searchMode === 'keyword' ? 'Search products...' : 'Ask a question...'"
        @keydown.enter="searchMode === 'ask' ? handleSubmit($event) : null"
      />
    </div>

    <!-- Keyword search results -->
    <div v-if="searchMode === 'keyword'">
      <ProductGrid :products="searchResults?.results ?? []" :loading="isSearching" />
    </div>

    <!-- AI answer -->
    <div v-else class="ai-answer">
      <div v-if="isAsking" class="ai-thinking">
        <SparklesIcon class="animate-pulse" />
        Analyzing your question...
      </div>
      <div v-else-if="aiAnswer" class="ai-response">
        <MarkdownRenderer :content="aiAnswer" />
      </div>
    </div>
  </div>
</template>
```

## 4. Search Result Enrichment with AI

Enhance search results with AI-generated summaries and relevance explanations.

```typescript
// server/api/search-enriched.post.ts
export default defineEventHandler(async (event) => {
  const { query } = await readBody(event)
  const results = await semanticSearch(query, { topK: 5 })

  // Generate relevance explanations in parallel for top results
  const enriched = await Promise.all(
    results.map(async (result) => {
      const { text: explanation } = await generateText({
        model: openai('gpt-4o-mini'),
        system: 'In one concise sentence, explain why this result is relevant to the query.',
        prompt: `Query: "${query}"\nResult: "${result.content.slice(0, 500)}"`,
        maxTokens: 80
      })
      return { ...result, relevanceExplanation: explanation }
    })
  )

  return { results: enriched }
})
```
