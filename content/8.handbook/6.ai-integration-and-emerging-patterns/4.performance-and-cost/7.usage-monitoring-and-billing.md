# Usage Monitoring & Billing

AI API costs scale linearly with usage. A single infinite loop or a rogue user can generate thousands of dollars in charges overnight. A production AI application requires robust telemetry, strict usage quotas, and automated cost billing.

## 1. Centralized Cost Telemetry

Never call an LLM without logging the usage. Use the Vercel AI SDK's `onFinish` callback to record token consumption for every request.

```typescript
// server/utils/llm-logger.ts
interface AIUsageLog {
  userId: string
  feature: string
  model: string
  inputTokens: number
  outputTokens: number
  costUsd: number
  durationMs: number
}

// Map models to their per-token cost (as of late 2024)
const getCost = (model: string, inputTokens: number, outputTokens: number) => {
  const rates: Record<string, { in: number, out: number }> = {
    'gpt-4o': { in: 0.000005, out: 0.000015 },
    'gpt-4o-mini': { in: 0.00000015, out: 0.0000006 },
    'claude-3-5-sonnet-20241022': { in: 0.000003, out: 0.000015 },
  }
  const rate = rates[model] || { in: 0, out: 0 }
  return (inputTokens * rate.in) + (outputTokens * rate.out)
}

export const logAIUsage = async (log: Omit<AIUsageLog, 'costUsd'>) => {
  const costUsd = getCost(log.model, log.inputTokens, log.outputTokens)
  
  // Asynchronously insert into your database (PostgreSQL, ClickHouse, etc.)
  await db.aiUsage.insert({ ...log, costUsd, createdAt: new Date() })
  
  return costUsd
}
```

```typescript
// Usage in your API route
const startTime = performance.now()
const result = await streamText({
  model: openai('gpt-4o-mini'),
  messages,
  onFinish: async ({ usage }) => {
    await logAIUsage({
      userId: event.context.userId,
      feature: 'chat_assistant',
      model: 'gpt-4o-mini',
      inputTokens: usage.promptTokens,
      outputTokens: usage.completionTokens,
      durationMs: performance.now() - startTime
    })
  }
})
```

## 2. Enforcing Quotas and Limits

Prevent abuse by enforcing hard limits on daily or monthly usage per user.

```typescript
// server/middleware/ai-quotas.ts
export default defineEventHandler(async (event) => {
  // Only apply to AI routes
  if (!event.path.startsWith('/api/ai/')) return

  const userId = event.context.userId
  const userPlan = event.context.userPlan // 'free' | 'pro'
  
  // Calculate usage for current month
  const currentMonth = new Date().toISOString().slice(0, 7) // '2024-10'
  const monthlyCost = await db.aiUsage
    .where({ userId, month: currentMonth })
    .sum('costUsd')

  // Enforce limits
  const limits = { free: 1.00, pro: 10.00 } // Max $ spend allowed
  const limit = limits[userPlan]

  if (monthlyCost >= limit) {
    throw createError({
      statusCode: 402, // Payment Required
      statusMessage: 'AI usage limit reached for current billing cycle.',
      data: { currentUsage: monthlyCost, limit }
    })
  }
})
```

## 3. Stripe Metered Billing (Pay-as-you-go)

If you are reselling AI access, standard subscriptions rarely work because power users will destroy your profit margins. Use Stripe Metered Billing to charge users for exactly what they use.

```bash
pnpm add stripe
```

```typescript
// server/utils/stripe-metering.ts
import Stripe from 'stripe'
const stripe = new Stripe(process.env.STRIPE_SECRET_KEY!)

export const reportUsageToStripe = async (stripeSubscriptionItemId: string, costUsd: number) => {
  // Convert dollars to cents (Stripe's base unit) or a custom token unit
  const units = Math.ceil(costUsd * 100) 
  
  await stripe.subscriptionItems.createUsageRecord(
    stripeSubscriptionItemId,
    {
      quantity: units,
      timestamp: Math.floor(Date.now() / 1000),
      action: 'increment',
    }
  )
}
```

## 4. Cost Anomaly Detection

Set up daily automated checks to catch runaway costs before the cloud provider sends you a bill.

```typescript
// server/cron/check-anomalies.ts
export const checkCostAnomalies = async () => {
  const yesterday = getYesterdayDate()
  const lastWeekAverage = await getAverageDailyCost(7)
  
  const yesterdayCost = await db.aiUsage
    .where('date', yesterday)
    .sum('costUsd')
    
  // Alert if yesterday's cost was 3x the recent average
  if (yesterdayCost > lastWeekAverage * 3 && yesterdayCost > 50) {
    await sendSlackAlert(`🚨 AI Cost Anomaly Detected! 
      Yesterday's spend: $${yesterdayCost.toFixed(2)}
      7-day average: $${lastWeekAverage.toFixed(2)}`
    )
  }
}
```

## 5. Crucial Provider Protections

No matter how good your application-level limits are, bugs happen. Always set hard limits at the provider level:

1. **OpenAI Platform:** Go to Settings → Billing → Usage limits. Set a hard cap (e.g., $100/mo) where API requests will instantly be rejected.
2. **Anthropic Console:** Go to Settings → Billing → Set a spend limit.
3. **AWS/GCP/Azure:** Set up budget alerts and auto-shutdown scripts.
